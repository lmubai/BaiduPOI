# BaiduPOI
百度地图poi 获取某一地区数据集成信息

本示例展示获取新疆地区，所有药店的详细 经纬度，地址，电话，药店名信息

抽象来讲，本示例可以应用到所有百度地图 POI 信息的抓取工作，（但是还是有些局限性问题：
百度地图返回的矩形区域总数！= 所有子区域的数据综合，会有一定量的数据丢失）
